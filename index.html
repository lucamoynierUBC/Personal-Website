Projects

Sound Board Desktop App

This was my firt ever coding project. The main motivation behind developing this app came from DJing. There was no fluid method to play short audi-bits of my favourite adlibs on any DJ software withought interrupting a mix between two songs. So instead of making a to-do list app(the seemingly popular project idea for most students in this course), I created a simple desktop soundboard. The soundboard plays adlibs, loops, and compilations.

I self learned the Java libraries Swing and AWT to build a GUI, and JSON to load and save loops and audiofiles. I also extensively documented every method and class, and included a JUnit suite with 100% code coverage for the back end. Don't mind the the gifs and images in the GUI. Some of them were just there to fill out some requirements for the course project.

Mold Art

Desktop app that that grows "mold" in an emergent pattern using simple rules of interactions. Each mold object is an array-like data structure that contains the x and y coordinates of the space occupied by the mold. I employed a composite design pattern allowing the mold to grow in a tree like structure.

Image Compression

Co-developed an image compression program with a partner in my Algorithms and Data Structures course. By using binary trees to represent PNG images, the program uses complex recursive algorithms for manipulating an image. The program can compress images and flip images horrizontally or verticaly by manipulating pointers.

The tree constructed from the image on the left has 57,344 leaf nodes, where on the right, the image obtained after compression contains 6,518 nodes. Each Node represents one pixel of the image.
